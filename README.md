## Т“бҙҮбҙҖбҙӣбҙңКҖбҙҮs

### Т“бҙҸКҖбҙЎбҙҖКҖбҙ…

- <b>Т“бҙҸКҖбҙЎбҙҖКҖбҙ… бҙҚбҙҮбҙ…ЙӘбҙҖ КҷбҙҮбҙӣбҙЎбҙҮбҙҮЙҙ бҙ„КңбҙҖЙҙЙҙбҙҮКҹs (sбҙҖбҙҚбҙҮ-бҙӣбҙҸ-sбҙҖбҙҚбҙҮ)</b>
- <b>КҷКҸбҙҳбҙҖss КҖбҙҮsбҙӣКҖЙӘбҙ„бҙӣбҙҮбҙ… / бҙҳКҖбҙҸбҙӣбҙҮбҙ„бҙӣбҙҮбҙ… бҙ„КңбҙҖЙҙЙҙбҙҮКҹs КҷКҸ бҙ…бҙҸбҙЎЙҙКҹбҙҸбҙҖбҙ… вҶ’ КҖбҙҮ-бҙңбҙҳКҹбҙҸбҙҖбҙ…</b>
- <b>бҙҖЙҙКҸбҙҸЙҙбҙҮ бҙ„бҙҖЙҙ КҹбҙҸЙўЙӘЙҙ бҙЎЙӘбҙӣКң бҙӣКңбҙҮЙӘКҖ бҙҸбҙЎЙҙ бҙӣбҙҮКҹбҙҮЙўКҖбҙҖбҙҚ бҙҖбҙ„бҙ„бҙҸбҙңЙҙбҙӣ вҖ” ЙӘЙҙКҹЙӘЙҙбҙҮ КҹбҙҸЙўЙӘЙҙ КҷбҙңбҙӣбҙӣбҙҸЙҙ бҙҸКҖ <code>/login</code></b>
- <b>бҙҳбҙҮКҖ-бҙңsбҙҮКҖ sбҙҸбҙңКҖбҙ„бҙҮ & бҙ…бҙҮsбҙӣЙӘЙҙбҙҖбҙӣЙӘбҙҸЙҙ sбҙҮбҙӣбҙӣЙӘЙҙЙўs</b>
- <b>бҙҖКҹКҹ Т“бҙҸКҖбҙЎбҙҖКҖбҙ…бҙҮбҙ… бҙ„бҙҖбҙҳбҙӣЙӘбҙҸЙҙs бҙҖКҖбҙҮ бҙҖбҙңбҙӣбҙҸбҙҚбҙҖбҙӣЙӘбҙ„бҙҖКҹКҹКҸ КҷбҙҸКҹбҙ…бҙҮбҙ… бҙңsЙӘЙҙЙў <code>&lt;b&gt;...&lt;/b&gt;</code> КңбҙӣбҙҚКҹ бҙӣбҙҖЙўs</b>

### бҙҖбҙңбҙӣбҙҸ-бҙҖбҙ„бҙ„бҙҮбҙҳбҙӣ

- <b>бҙҖбҙңбҙӣбҙҸ-бҙҖбҙҳбҙҳКҖбҙҸбҙ бҙҮs бҙҮбҙ бҙҮКҖКҸ бҙҠбҙҸЙӘЙҙ КҖбҙҮЗ«бҙңбҙҮsбҙӣ бҙӣКңбҙҮ КҷбҙҸбҙӣ sбҙҮбҙҮs (бҙ„КңбҙҖЙҙЙҙбҙҮКҹ бҙҸКҖ ЙўКҖбҙҸбҙңбҙҳ)</b>
- <b>бҙҸбҙҳбҙӣЙӘбҙҸЙҙбҙҖКҹ бҙЎбҙҮКҹбҙ„бҙҸбҙҚбҙҮ бҙҳбҙҚ бҙЎЙӘбҙӣКң бҙҳКҹбҙҖбҙ„бҙҮКңбҙҸКҹбҙ…бҙҮКҖs (<code>{mention}</code>, <code>{first_name}</code>, <code>{username}</code>, <code>{chat_title}</code>, <code>{chat_link}</code>, <code>{user_id}</code>)</b>
- <b>бҙҳбҙҮКҖ-бҙ„КңбҙҖбҙӣ бҙ„бҙңsбҙӣбҙҸбҙҚ бҙЎбҙҮКҹбҙ„бҙҸбҙҚбҙҮ бҙҚбҙҮssбҙҖЙўбҙҮ + бҙҸЙҙ / бҙҸТ“Т“ бҙӣбҙҸЙўЙўКҹбҙҮ (бҙ„КңбҙҖбҙӣ бҙҖбҙ…бҙҚЙӘЙҙs)</b>
- <b>бҙҸбҙЎЙҙбҙҮКҖ-бҙҸЙҙКҹКҸ <code>/stats</code>, <code>/chats</code>, <code>/broadcast</code></b>
- <b>Т“КҹбҙҸбҙҸбҙ…бҙЎбҙҖЙӘбҙӣ-sбҙҖТ“бҙҮ бҙҖЙҙбҙ… бҙҖбҙңбҙӣбҙҸ-бҙ„КҹбҙҮбҙҖЙҙs бҙ…бҙҮбҙҖбҙ… бҙңsбҙҮКҖs бҙҸЙҙ КҷКҖбҙҸбҙҖбҙ…бҙ„бҙҖsбҙӣ</b>

## бҙҳКҖбҙҸбҙҠбҙҮбҙ„бҙӣ sбҙӣКҖбҙңбҙ„бҙӣбҙңКҖбҙҮ

```
.
в”ңв”Җв”Җ miko.py              # entry point: python3 miko.py
в”ңв”Җв”Җ config.py            # environment configuration
в”ңв”Җв”Җ requirements.txt
в”ңв”Җв”Җ database/
в”Ӯ   в””в”Җв”Җ db.py            # mongodb (motor) wrapper вҖ” users, sessions, chats, settings, counters
в””в”Җв”Җ plugins/
    в”ңв”Җв”Җ start.py         # /start, /help вҖ” photo + Login/Logout buttons
    в”ңв”Җв”Җ logins.py        # /login, /logout, /cancel + button-driven login flow
    в”ңв”Җв”Җ settings.py      # /settings, /setsource, /setdest, /clearsettings
    в”ңв”Җв”Җ forward.py       # /forward, /stop вҖ” clones a message range
    в”ңв”Җв”Җ accept.py        # auto-accept ChatJoinRequest handler + welcome PM
    в”ңв”Җв”Җ welcome.py       # /setwelcome, /clearwelcome, /togglewelcome, /welcome
    в”ңв”Җв”Җ stats.py         # /stats, /chats вҖ” owner only
    в””в”Җв”Җ broadcast.py     # /broadcast вҖ” owner only
```

## sбҙҮбҙӣбҙңбҙҳ

1. <b>бҙ„КҖбҙҮбҙҖбҙӣбҙҮ бҙҖ КҷбҙҸбҙӣ бҙ ЙӘбҙҖ</b> [@BotFather](https://t.me/BotFather) <b>бҙҖЙҙбҙ… ЙўбҙҮбҙӣ бҙӣКңбҙҮ</b> <code>BOT_TOKEN</code><b>.</b>
2. <b>ЙўбҙҮбҙӣ</b> <code>APP_ID</code> <b>бҙҖЙҙбҙ…</b> <code>API_HASH</code> <b>Т“КҖбҙҸбҙҚ</b> <https://my.telegram.org><b>.</b>
3. <b>sбҙҮбҙӣ бҙӣКңбҙҮ бҙҮЙҙбҙ ЙӘКҖбҙҸЙҙбҙҚбҙҮЙҙбҙӣ бҙ бҙҖКҖЙӘбҙҖКҷКҹбҙҮs (бҙҸКҖ бҙҮбҙ…ЙӘбҙӣ бҙӣКңбҙҮ бҙ…бҙҮТ“бҙҖбҙңКҹбҙӣs ЙӘЙҙ</b> <code>config.py</code><b>):</b>

   ```
   BOT_TOKEN=...
   APP_ID=...
   API_HASH=...
   OWNER=your_telegram_username
   OWNER_ID=your_numeric_id
   MONGO_URI=mongodb+srv://...
   DATABASE_NAME=Forward
   START_PIC=https://graph.org/file/...   # optional
   ACCEPT_DELAY=0                          # optional, seconds before each approve
   DEFAULT_WELCOME=...                     # optional, default welcome PM template
   ```

4. <b>ЙӘЙҙsбҙӣбҙҖКҹКҹ бҙ…бҙҮбҙҳбҙҮЙҙбҙ…бҙҮЙҙбҙ„ЙӘбҙҮs:</b>

   ```bash
   pip install -r requirements.txt
   ```

5. <b>КҖбҙңЙҙ бҙӣКңбҙҮ КҷбҙҸбҙӣ:</b>

   ```bash
   python3 miko.py
   ```

6. <b>Т“бҙҸКҖ бҙҖбҙңбҙӣбҙҸ-бҙҖбҙ„бҙ„бҙҮбҙҳбҙӣ вҖ” бҙҖбҙ…бҙ… бҙӣКңбҙҮ КҷбҙҸбҙӣ бҙҖs бҙҖЙҙ бҙҖбҙ…бҙҚЙӘЙҙ ЙӘЙҙ КҸбҙҸбҙңКҖ бҙ„КңбҙҖЙҙЙҙбҙҮКҹ / ЙўКҖбҙҸбҙңбҙҳ бҙЎЙӘбҙӣКң "бҙҖбҙ…бҙ… бҙҚбҙҮбҙҚКҷбҙҮКҖs" бҙҳбҙҮКҖбҙҚЙӘssЙӘбҙҸЙҙ, бҙҖЙҙбҙ… бҙҮЙҙбҙҖКҷКҹбҙҮ "бҙҖбҙҳбҙҳКҖбҙҸбҙ бҙҮ ЙҙбҙҮбҙЎ бҙҚбҙҮбҙҚКҷбҙҮКҖs".</b>

## КҹбҙҸЙўЙӘЙҙ Т“КҹбҙҸбҙЎ

1. <b>бҙӣбҙҖбҙҳ бҙӣКңбҙҮ КҹбҙҸЙўЙӘЙҙ КҷбҙңбҙӣбҙӣбҙҸЙҙ бҙңЙҙбҙ…бҙҮКҖ</b> <code>/start</code><b>, бҙҸКҖ sбҙҮЙҙбҙ…</b> <code>/login</code><b>.</b>
